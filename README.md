# Resma CLI

This is a CLI :)

# Folder structure

|-- config.toml
|-- content
|-- templates
|-- styles
|-- static

In the content folder, a page should be a new MarkDown file, and a section a new folder.

├── content/
│   └── blog/
│       ├── _index.md
│       ├── first.md
│       └── second.md

After the build, a folder called "public" will be created with your nice blog!
